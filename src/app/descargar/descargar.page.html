Proceso de Instalación

1.Descarga e instala

Descarga la APK de Magistv directamente a tu Celular Android o bien a una memoria USB para instalarla en una TV box Android.




2. Inicia sesión

Cuando termine realizar al pago ， utiliza el usuario y la contraseña . Inicia sesión en tu Celular o TV box Android para activar el servicio.




3. Dispositivos eligidos

* 3 dispositivos android tv box
* 2 dispositivos android tv box + 1 celular
* 3 celulares , o tablets Android .

Nota : Dispositivos incluyen android celular, tablets android, android tv box, android tv, smart tv android, etc

MagisTV conexión válida para tres dispositivos de forma simultánea , e exclusivos para dispositivos en Android más de 5.1 y superior ( Android TV Box,Android Tv, etc...)

Para utilizar el servicio de manera optima se necesita una conexión de Internet estable, con una velocidad de 2 MB internacional como mínimo. (Notas : Algunos canales de FHD y vod de 4K necesita un buen internet estable y un dispositivo con alta configuración, muchas gracias por entender. )
